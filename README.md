# 🌍 Dual City Weather & Air Quality Dashboard

## 📌 Project Overview
This project is a single-page web application that allows users to compare real-time weather conditions and air quality between two cities simultaneously.

It uses OpenWeatherMap APIs along with jQuery and AJAX to fetch and display live data dynamically.

---

## 🚀 Features

- 🔍 Search and compare two cities
- 🌡 Real-time temperature and humidity
- 🌫 Air Quality Index (AQI) with PM2.5 data
- 🎨 Dynamic background color based on AQI levels
- ⚡ Fast data fetching using AJAX
- ❌ Error handling for invalid city inputs
- 🔄 Chained API calls (Geocoding → Weather → Air Quality)

---

## 🛠 Technologies Used

- HTML5
- CSS3
- Bootstrap
- jQuery
- AJAX
- JSON
- OpenWeatherMap API

---

## 🔗 APIs Used

- Geocoding API (City → Latitude & Longitude)
- Current Weather API
- Air Pollution API
- Forecast API (optional)
